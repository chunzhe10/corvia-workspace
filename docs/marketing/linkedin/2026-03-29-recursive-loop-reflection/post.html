Every time I've changed projects or roles, I spend too much time rebuilding my environment, my workflow, my tools. I wanted something portable. Something I could carry wherever I go. So about 30 days ago, I started building corvia. An organizational memory system for AI agents, written in Rust. I wanted to learn generative AI firsthand by building in it, and I wanted to build something real. AI agents are developing memory. CLAUDE.md, Cursor rules, Copilot. These work for individual developers. But they're personal memory. What happens when agents need to share knowledge across repos? When decisions need to be tracked over time? That's organizational memory. That's the layer corvia is building. But there was a catch. The memory system didn't exist yet. So I recorded every design decision in markdown files by hand. The plan was to migrate everything once the tool could hold its own knowledge. Day 1 was markdown files and Obra's superpowers plugin for brainstorming. No review gate. No agent coordination. Some early decisions got missed. The first real milestone was getting corvia to ingest its own decisions. Once those files migrated in, the development loop started forming. The process got better as the product got better. Then came the first test outside corvia's own development. An interview take-home. Same approach. Structured decision recording. Querying past decisions before building. Finished in less than 24 hours. That was the moment it started feeling like infrastructure I'd actually carry forward. Not everything shipped perfectly. Some recent capabilities haven't been fully used yet. Getting hands-on with AI development is fundamentally different from reading about it. You have to feel it break, watch it recover, and build intuition from there. I'm building this on a Claude Max 5 plan. Not cheap when you're unemployed, but I'm committed. When they ran the 2x token limit promotion, I was out diving and missed most of it. Timing. hahaha Claude shipped memory, Agent Teams, and session persistence. I find that exciting, not threatening. Personal memory is becoming table stakes. What's still open is the organizational layer. I think these tools could work really well together. 30 days: 700+ commits across both repos, 12 releases, 10 Rust crates, 1,100+ tests, 100+ decisions recorded, 21 MCP tools.
