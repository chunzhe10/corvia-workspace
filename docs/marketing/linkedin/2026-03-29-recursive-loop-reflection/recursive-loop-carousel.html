The Bootstrap
Day 1: No Tool Yet
Building a memory system that can't remember anything. Yet.

Markdown files

Asked Claude to save every decision in a specific format. Brainstorms, trade-offs, architecture calls. All by hand. The plan: migrate it all once the tool is ready.

Obra's superpowers plugin

The brainstorming skill structured every design session. Explore, clarify, converge. This was the process from day 1. The rest came later.

First self-ingest

corvia ingests its own decisions. The markdown files migrate in. Now the tool can search its own history. The loop starts forming.

The loop gets better

5-persona review, session history, agent coordination. The process and the product grew together.

The Real Test
Beyond Dogfooding

Interview take-home

First time using corvia outside its own development. Structured decision recording. Querying past decisions before building. I didn't expect it to be ready.

< 24 hours

The methodology transferred. That was the moment it stopped being just a portfolio project.

The honest part

Not everything shipped perfectly. A lot of recent features haven't been fully used yet. Agent teams, structural code intelligence, session history. Built and tested, but I'm still internalizing them.

Getting hands-on with AI development is fundamentally different from reading about it. You have to feel it break, watch it recover, and build intuition from there.

Key Takeaways
What I Learned
🔁

Dogfood as early as possible

Race to self-host. The sooner the tool builds itself, the sooner the loop reveals what's actually needed.

🖐️

Hands-on beats reading

You can't understand AI development from blog posts. You have to build, feel it break, and build intuition firsthand.

🌊

Grab every emerging capability

Claude shipped Agent Teams. I built an adapter. GitNexus showed structural intelligence. I started building that too. AI moves fast. Ride the wave.

🌐

Personal memory is table stakes

CLAUDE.md, Cursor rules, Copilot memory. What's still open is the organizational layer. Memory across repos, over time, between agents.
